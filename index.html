About Me

I have always been interested in code ;

Ever since my first days in the retail sector, I have seen the Excel sheets, the KPI dashboards and the Customer Management software.
And it has left me wanting....

I suppose it all started with Excel VBA

In every job I have had to date, I have always tried to get the most out of the Microsoft Office Package;
Everyone uses it, its easy to understand, and fairly easy to customise.
But, as with most technologies, keeping it up-to-date, easy-to-use and fully-functional seems to end up being a mammoth task.
I have made many a custom dashboard, weekly calender and custom datastore with Excel
And although the language is restrictive and the syntax strict.
It has provided me a baseline of programming concepts, from ;
  • Custom UI design
  • Handling Databases
  • Separation of concerns
  • Basic data analysis
    • And most Importantly...
  • Error Handling

    • To Web Development.. ;

    Being a 'Forever Student', I quickly realised i could accomplish everything I needed with Web Technologies _>

    Need a great UI ? HTML + css
    Need to do something complex ? Javascript
    Need a custom database ? MongoDB
    Need a way to customise your content ? Node.js

    At first i thought it was going to be PHP...

    Being one of the most popular languages on the internet today with age on its side and a huge community.
    I muddled my way through a few tutorials online; Installed XAMP, created my own local server and had created my first page.
    I liked the "Just put it together... it will work" philosphy.
    But as i went deeper into the rabbit-hole, i found myself being drawn more and more to Javascript as a Full-Stack solution.

    ...But now im thinking the MEVN Stack

    "One language to Rule them All" __ My Journey Continues...
